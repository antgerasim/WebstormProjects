Handling External Changes

Value:

Now, when you click on one of the button, the select box will change its value

Use the jqLite 'on' method to register a handler function for the click event on the button elements in the directive template. When the user clicks one of the buttons, the NgModel controller gets notified by calling the $setViewValue method

scope.$apply(function () { ctrl.$setViewValue(event.target.innerText); });

The scope.$apply method is used to push updates into the data model. The scope will execute the function and then update its state; using a function allows me to notify the NgModel controller of the change and have the scope update in a single step.

To update the data-bound value, I call the $setViewValue, which accepts the new value as its argument.

For this example, I get the value from the text content of the button that has been clicked, such that clicking the Yes button causes the dataValue property to be set to Yes.
