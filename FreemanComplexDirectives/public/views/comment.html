Setting the restrict Option in the directives.html File

Products

I apply the directive as an unordered-list element, which I configure using attributes. This requires me to make a change to the link function for the directive because the source of the data has to be defined with a new attribute. I have selected the name list-source for the new attribute, and you can see how I check for the value of the attribute if there is no unordered-list attribute value available:

<div unordered-list="products" list-property="price | currency"></div>
